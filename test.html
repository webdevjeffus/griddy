This section includes main content and a sidebar

The content in this section contains an <article> and an <aside>. On phones, both elements are full-width, with the article first and the aside second. This is accomplished by assigning both elements the ph-col-12 class.

On tablets and larger screen-sizes, the two elements are arranged in a traditional main-text/sidebar layout. This is accomplished by assigning the article element the tab-col-8 class, while the aside element gets the tab-col-4 class. Since no column-width classes are assigned to either element for desktop screen-sizes, the column-width classes for tablets stands for all larger screens.
